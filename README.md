# javascript-practice

『Do it! 자바스크립트』 를 공부하며 작성한 자바스크립트 연습 코드들 정리


---

## 학습 범위

- **6장**: 여러 자료를 한꺼번에 담는 객체
- **7장**: 배열을 쉽게 다루는 Array 객체
- **8장**: 웹 문서를 다루는 방법 (DOM)
- **9장**: 폼과 자바스크립트
- **10장**: 웹 브라우저를 다루는 방법 (BOM)

---

## 구성 내용

- 객체 생성 및 활용
- 배열 데이터 처리
- DOM 요소 선택 및 이벤트 처리
- 요소 생성 / 추가 / 삭제
- 폼 입력값 검증
- 이미지 변경 및 UI 동작 구현
